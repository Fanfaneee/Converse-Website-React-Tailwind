## Installation

Clone the project 
```bash
  git clone https://github.com/Fanfaneee/Converse-Website-React-Tailwind.git
```
Enter the project
```bash
 cd Converse-Website-React-Tailwind
```
Install npm
```bash
  npm install 
```
Run the project 
```bash
  npm run dev 
```
